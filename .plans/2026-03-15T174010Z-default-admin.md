title: Default admin user at startup
created: 2026-03-15T17:40:10Z
context: default-admin
description: Create a default admin user on startup if no ADMIN user exists in DB.
---

assumptions:
- Runs after Flyway migrations via ApplicationRunner.
- Username is always "admin".
- Password is randomly generated (UUID or SecureRandom hex) on first startup.
- Generated password is printed to console (System.out or log.warn) so operator can log in.
- Password is NOT stored in config — generated at runtime, encoded, saved to DB.
- Idempotent: checks USER_ROLES for any ADMIN row; if one exists, skips entirely.
- Uses existing UserManagement.register() + setRole().
- DuplicateUsernameException catch: "admin" user may exist without ADMIN role — handle by
  skipping register() and just calling setRole().

phases:
  - phase: 1 — DefaultAdminInitializer bean
    files:
      - src/main/java/simple/simple_webapp/user/DefaultAdminInitializer.java  (NEW)
        - @Component, implements ApplicationRunner
        - Inject DSLContext, UserManagement
        - run():
            boolean adminExists = dsl.fetchExists(
              dsl.select().from(USER_ROLES)
                 .where(USER_ROLES.ROLE.eq(UserRole.ADMIN.name()))
            );
            if (adminExists) return;
            String password = generatePassword(); // UUID.randomUUID().toString()
            try {
              userManagement.register("admin", password);
            } catch (DuplicateUsernameException ignored) {}
            var user = dsl.selectFrom(USERS)
                         .where(USERS.USERNAME.eq("admin"))
                         .fetchOne();
            userManagement.setRole(user.getId(), UserRole.ADMIN);
            System.out.println("===========================================");
            System.out.println("Default admin created. Username: admin");
            System.out.println("Password: " + password);
            System.out.println("===========================================");

  - phase: 2 — Test
    files:
      - src/test/java/simple/simple_webapp/user/DefaultAdminInitializerTests.java  (NEW)
        - @SpringBootTest, @Testcontainers, @Import(UserTestConfiguration.class)
        - Inject UserManagement, DefaultAdminInitializer
        - adminUserExistsAfterStartup():
            details = userManagement.loadUserByUsername("admin");
            assertThat(details.getAuthorities()).extracting(Object::toString).contains("ROLE_ADMIN");
        - secondRunIsIdempotent():
            call initializer.run(null) a second time;
            count rows in USER_ROLES where role = 'ADMIN';
            expect exactly 1.
